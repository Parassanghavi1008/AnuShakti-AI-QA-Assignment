# AnuShakti AI – QA Assignment

This repository contains the **QA testing work** performed for the AnuShakti AI application, an AI-powered technical assistant for the Atomic Energy Regulatory Board (AERB) documents.

## Project Overview
AnuShakti AI helps users retrieve information from AERB publications with:
- Accurate answers with source citations
- Non-hallucination behavior
- Handling of text, numerical, and image-based queries

## Repository Contents
- `Test_Scenarios/` – Top 3 critical test scenarios identified
- `Test_Cases/` – Detailed test cases with execution status
- `Defect_Reports/` – Defects logged with steps, severity, and priority
- `Test_Summary/` – 1-page summary with risks, quality assessment, and release recommendation
- `Screenshots/` – Relevant screenshots (errors, outputs, etc.)

## Key Highlights
- Focused on **core product functionality and risk-based testing**
- Executed **edge cases and failure scenarios** to validate stability
- Documented **release-blocking defects** and proposed mitigation
- Followed **real QA documentation standards**
